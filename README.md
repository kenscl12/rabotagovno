# RabotaGovnoClone (Blazor, .NET 8)

Минимальный клон страницы отзыва (маршрут `/main/view/all/{id}`) + список "новые" и форма добавления комментариев/отзывов.

## Запуск

1) Установите .NET SDK 8: https://dotnet.microsoft.com/

2) В папке проекта:

```bash
dotnet restore
dotnet run
```

3) Откройте в браузере URL из консоли (обычно `http://localhost:5090`).

## Что внутри

- `/` — новые отзывы
- `/best` — лучшие (по рейтингу)
- `/add` — добавить отзыв
- `/main/view/all/16626` — пример страницы отзыва, как в референсе
